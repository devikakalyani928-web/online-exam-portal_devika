PROJECT DATABASE SCHEMA
========================

All collections are stored in MongoDB via Mongoose. Each collection below lists its fields in a tabular format with:
- Field Name
- Data Type
- Constraints
- Required
- Description

------------------------------------------------------------
COLLECTION: users
------------------------------------------------------------

Field Name      | Data Type | Constraints                                                            | Required | Description
--------------- | --------- | ---------------------------------------------------------------------- | -------- | -------------------------------------------------------------
_id             | ObjectId  | Primary Key, auto-generated by MongoDB                                | Yes      | Unique identifier for each user document.
username        | String    | Unique, trimmed; regex: /^[a-zA-Z0-9._]+$/ (letters, numbers, underscore, dot only) | Yes      | Login username; used for identification in the system.
full_name       | String    | Trimmed; regex: /^[a-zA-Z\s]+$/ (letters and spaces only); min length: 2 | Yes      | Full name of the user (letters and spaces, min length 2).
email           | String    | Unique, lowercase, trimmed; valid email format                         | Yes      | User email address used for login and communication.
password        | String    | Hashed using bcrypt before save; min length: 6 characters             | Yes      | Encrypted password (never stored in plain text).
role            | String    | Enum: 'System Admin', 'Exam Manager', 'Question Manager', 'Result Manager', 'Student'; default 'Student' | No (default) | Role used for role-based access control and dashboards.
createdAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Document creation timestamp.
updatedAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Last update timestamp.

------------------------------------------------------------
COLLECTION: exams
------------------------------------------------------------

Field Name      | Data Type | Constraints                                                            | Required | Description
--------------- | --------- | ---------------------------------------------------------------------- | -------- | -------------------------------------------------------------
_id             | ObjectId  | Primary Key, auto-generated by MongoDB                                | Yes      | Unique identifier for each exam.
exam_name       | String    | Trimmed; regex: /^[a-zA-Z\s]+$/ (letters and spaces only, no numbers or special characters) | Yes      | Name/title of the exam.
start_time      | Date      | Must be >= current date/time (date portion cannot be before today)   | Yes      | Date and time when the exam becomes available to students.
end_time        | Date      | Must be > start_time (end time must be greater than start time)      | Yes      | Date and time when the exam closes.
duration        | Number    | Positive integer (minutes); min value: 1                               | Yes      | Duration of the exam in minutes (can be set in minutes or hours; stored as minutes).
created_by      | ObjectId  | Ref: users._id                                                        | Yes      | User (Exam Manager) who created the exam.
is_active       | Boolean   | Default false                                                        | No (default) | Controls whether students can see/attempt the exam.
createdAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Document creation timestamp.
updatedAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Last update timestamp.

------------------------------------------------------------
COLLECTION: questions
------------------------------------------------------------

Field Name      | Data Type | Constraints                                                            | Required | Description
--------------- | --------- | ---------------------------------------------------------------------- | -------- | -------------------------------------------------------------
_id             | ObjectId  | Primary Key, auto-generated by MongoDB                                | Yes      | Unique identifier for each question.
exam_id         | ObjectId  | Ref: exams._id                                                        | Yes      | The exam to which this question belongs.
question_text   | String    | Trimmed; not empty                                                    | Yes      | Text/content of the question.
option1         | String    | Not empty                                                              | Yes      | First answer option.
option2         | String    | Not empty                                                              | Yes      | Second answer option.
option3         | String    | Not empty                                                              | Yes      | Third answer option.
option4         | String    | Not empty                                                              | Yes      | Fourth answer option.
correct_option  | Number    | Enum: 1, 2, 3, 4                                                      | Yes      | Indicates the correct option number (1–4).
created_by      | ObjectId  | Ref: users._id                                                        | Yes      | User (Question Manager) who created the question.
createdAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Document creation timestamp.
updatedAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Last update timestamp.

------------------------------------------------------------
COLLECTION: examattempts
------------------------------------------------------------

Field Name      | Data Type | Constraints                                                            | Required | Description
--------------- | --------- | ---------------------------------------------------------------------- | -------- | -------------------------------------------------------------
_id             | ObjectId  | Primary Key, auto-generated by MongoDB                                | Yes      | Unique identifier for each exam attempt.
exam_id         | ObjectId  | Ref: exams._id                                                        | Yes      | Exam being attempted.
student_id      | ObjectId  | Ref: users._id                                                        | Yes      | Student user who is attempting the exam.
start_time      | Date      | Default: current date/time                                            | Yes      | Time when the attempt started.
end_time        | Date      |                                                                        | No       | Time when the attempt finished (set when student completes or time expires).
total_score     | Number    | Default 0                                                             | No (default) | Total marks obtained in this attempt.
completed       | Boolean   | Default false                                                        | No (default) | Whether the attempt has been submitted/completed.
is_passed       | Boolean   | Default false                                                        | No (default) | Pass/fail result based on pass threshold (e.g., 40%).
createdAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Document creation timestamp.
updatedAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Last update timestamp.

------------------------------------------------------------
COLLECTION: studentanswers
------------------------------------------------------------

Field Name      | Data Type | Constraints                                                            | Required | Description
--------------- | --------- | ---------------------------------------------------------------------- | -------- | -------------------------------------------------------------
_id             | ObjectId  | Primary Key, auto-generated by MongoDB                                | Yes      | Unique identifier for each student answer record.
attempt_id      | ObjectId  | Ref: examattempts._id                                                 | Yes      | Exam attempt to which this answer belongs.
question_id     | ObjectId  | Ref: questions._id                                                    | Yes      | Question answered by the student.
selected_option | Number    | Nullable; must be null/undefined or 1, 2, 3, 4; default null; custom validator: value === null || value === undefined || [1, 2, 3, 4].includes(value) | No       | Option chosen by the student (null means unanswered / skipped).
is_correct      | Boolean   |                                                                        | Yes      | Indicates whether the selected option is correct.
createdAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Document creation timestamp.
updatedAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Last update timestamp.

------------------------------------------------------------
COLLECTION: feedbacks
------------------------------------------------------------

Field Name      | Data Type | Constraints                                                            | Required | Description
--------------- | --------- | ---------------------------------------------------------------------- | -------- | -------------------------------------------------------------
_id             | ObjectId  | Primary Key, auto-generated by MongoDB                                | Yes      | Unique identifier for each feedback entry.
student_id      | ObjectId  | Ref: users._id                                                        | Yes      | Student who submitted the feedback.
message         | String    | Trimmed                                                               | Yes      | Feedback text provided by the student.
reply           | String    | Trimmed; default null                                                 | No       | Optional reply from an admin/result manager.
status          | String    | Enum: 'Pending', 'Replied'; default 'Pending'                         | No (default) | Status of the feedback resolution.
createdAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Document creation timestamp.
updatedAt       | Date      | Auto-set by Mongoose timestamps                                      | Yes      | Last update timestamp.
